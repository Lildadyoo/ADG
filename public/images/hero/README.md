# Hero Images Directory

## Quick Start
Just drop your hero images here with these exact names:
- `hero-background.jpg` - Background image (1920x1080px recommended)
- `hero-featured.jpg` - Featured image for desktop (600x400px recommended)

That's it! The website will automatically use these images. If the files are missing, placeholder images will show instead.

## Image Specifications

### hero-background.jpg
- **Size**: 1920x1080px (or larger, landscape orientation)
- **Purpose**: Full-width background behind hero section
- **Display**: Shown at 20% opacity with gradient overlay
- **Content**: Should show community members, development work, or positive impact scenes

### hero-featured.jpg
- **Size**: 600x400px or larger (landscape orientation)
- **Purpose**: Featured image displayed on desktop in hero section right column
- **Display**: Full color with white border and shadow
- **Content**: Should showcase ADG's work, community engagement, or program impact

## File Formats
- Supported: JPG, PNG, WebP
- Recommended: JPG for best compatibility
- Next.js will automatically optimize these images
